# Četník - Počítadlo písmen 🚨

**Četník** je jednoduchá, rychlá a přehledná webová aplikace pro podrobnou analýzu textu. Dokáže spočítat nejen znaky a slova, ale zaměřuje se i na detailní rozbor diakritiky, symbolů a číslic.

Aplikaci najdete online na: **[nastroj.github.io/cetnik](https://nastroj.github.io/cetnik/)**

## Hlavní funkce
- 🔤 **Komplexní statistiky:** Okamžitý přehled o celkovém počtu znaků, písmen, symbolů a číslic.
- ´ **Analýza diakritiky:** Unikátní funkce, která počítá háčky a čárky v textu.
- ➖ **Odečítání znaků:** Možnost definovat specifické znaky, které se mají z celkového výsledku odečíst (včetně inteligentního rozlišení velkých/malých písmen).
- 📁 **Nahrávání souborů:** Můžete analyzovat text přímo vložením nebo nahráním souboru.
- 📊 **Detailní výpisy:** Rozdělení výsledků do kategorií (Vše, Písmena, Diakritika, Symboly, Čísla) pro maximální přehled.
- 🌙 **Dark Mode:** Podpora tmavého režimu pro šetření vašich očí.

## Jak to funguje
Aplikace běží kompletně v prohlížeči (client-side), což znamená, že váš text nikam neposílá a je v bezpečí. Analýza probíhá v reálném čase během psaní.

## Technologie

- CSS3 (Tailwind CSS)
